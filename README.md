# Sistema de Chamados - Laravel

Sistema de chamados para suporte técnico, desenvolvido com Laravel, MySQL e Tailwind.

## 🚀 Funcionalidades

- Criar chamados
- Editar chamados
- Listar chamados
- Excluir chamados
- Status: aberto, em andamento, resolvido, fechado
- Prioridade: baixa, média, alta
- Atribuição de responsável

## 🛠 Tecnologias

- Laravel
- PHP
- MySQL
- Tailwind CSS

## 💡 Objetivo

Projeto desenvolvido para estudo prático de CRUD e organização de sistema de suporte.
